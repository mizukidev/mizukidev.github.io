Win10にPythonDjangoとVScodeを入れてみた
Created 2020-06-28|WEB開発Win
Post View:

ＷＥＢサービス開発を始めたい方にむけて、快適に動く環境を目指してます　よろしくお願いします

作業環境
・Windows10 Pro
・VisualStudioCode
・Python(64bit) 3.6.5
・Django 2.0.6

構築する開発環境について

Python Djangoを使って手軽にＷＥＢサービスを開発したい　でも重たくてめんどうなものはいや　WEB開発だけに集中したい　そんな感じで作成しました

初心者の方でもとっつきやすくするため、仮想環境やＷＳＬを使わず、winに直接環境を構築するようにしました　

自分の環境が壊れたときの覚え書きと、開発仲間が増えたときに、まず最初にとりあえずこれやっての資料に使えたらないいなと思います　

まずは構築しやすい Python+Djangoから
（Railsの手順も作りました　こちらです)

pythonインストール

https://www.python.org/downloads/windows/
python 公式サイトにアクセスします Downloadからwindowsをクリックします

※デフォルトが32bitなのですが、64bitを選ぶほうが後々いいです

downloadのリストが出てきますので、その中から3.6.5を選び　インストーラーを選択してダウンロードしてください

クリックして起動

pathにチェックして、カストマイズをクリック

nextをクリック（※pipも同時インストールされます）

install all userにチェック、pathは空白なしのフォルダ名　installを押下

closeを押下だけでよいです

pathの文字列制限260文字を解除するそうです

コマンドラインで djangoインストール

#VisualStudioCodeのセットアップ
ダウンロードして、インストール

vscode上で拡張機能をインストールしておく

japanese language pack　と　python　を入れておく

#あとgitも入れといてください
ダウンロードして、インストール
https://gitforwindows.org/

githubというサイトでサインインしておく

#バージョンなどの確認

以上です
