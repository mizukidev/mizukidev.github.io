githubでスター数の多いソースの探し方
Created 2020-06-28|github
Post View:

githubの重要性

githubでスター数が多いソースは非常に利用価値があります

無料で効果されているので使わない手はありません

githubを使ってるかどうかは、プログラマーのスキル・成長に非常に大きく関わります

githubには良質なコードがたくさんあり、良質なプログラマーを育てる環境があります

検索の方法

https://github.com/ の検索ボックスで以下のように入力します

探したいキーワード stars:>1

たとえばCRUD(生成 読取 更新 削除)のサンプルを探したい場合はこんな感じで使います
CURD stars:>1 と入力すると https://github.com/search?q=crud+stars%3A%3E1

右のsortをクリックするとスター順に変えれます

urlに　&s=stars&o=desc　をつけるとスターでソートできます
https://github.com/search?q=crud+stars%3A%3E1&s=stars&o=desc

使い方の例

検索の例 説明 urlクエリー
crud stars:>=100 CRUDを含むスター100以上 https://github.com/search?q=crud+stars%3A%3E100
crud language:python stars:>=100 CRUDを含むpython言語のスター100以上 https://github.com/search?q=crud+language%3Apython+stars%3A%3E%3D100
crud stars:>=100 created:>2016-04-01 crudを含むスター100以上で作成日2016年4/1以降のもの https://github.com/search?q=crud+stars%3A%3E%3D100+created%3A%3E2015-04-01

githubの検索の仕様はすぐにわかると思いますが使えそうなのが
q=starts: created: pushed: language:
s=stars updated
type=epository ,code

などなど

githubのヘルプに仕様がありますので詳しくはこちらを
https://help.github.com/articles/understanding-the-search-syntax/
