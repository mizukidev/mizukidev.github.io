静的ブログなら、初年度780円２年目以降1180円

.coｍのお値段 １年目 ２年目以降 アフィ報酬 ２年費用合計
エックスドメイン（wordpress専用） 780円 1180円 -375円 1585円
お名前.com 760円 1280円 -456円 1584円
ムームドメイン（高いです） 1160円 1480円 -500円 2140円

静的ブログにしますので、お名前.com以外ないです　でも年間経費ですよこれ　安いでしょ

ほんとにドメイン代金以外の経費は必要なし　これほど低資金のビジネスは他にないですね。

さらにＡ８でお得なキャンペーンを使う

あなたはA８の自己アフィリエイト使っていますか？使えるキャンペーンは使いましょう

ちょっぴりお得　時期によって違いますが、だいたいいつもこんな感じです　

  • お名前.com
    成果報酬　600円/個（1ドメインの注文金額が1,000円未満は、注文金額の60％）
    お名前.comへのドメイン移管　50円
    760 * 60% = 456円

  • エックスドメイン
    成果報酬　ドメイン取得　 500円 or 購入50％（1000円(税抜)未満のドメイン）
    500 円 or (780 * 50%) = 375円

ワードプレスの年間維持費は１万円を軽く超えます

ワードプレスの場合

.coｍのお値段 初期経費 年間経費 年間経費計
エックスサーバーX10プラン 3300円 1100円 x 12=13200円 16500円
conoha WING 990円 x 12=11800円 11800円

超えますよね？普通に超えます　現実を直視しましょう　初年度はほぼマイナスですので

で、マルチでサイトを登録すればいいのですが、それもすごく重たくなります　そうなるとスマホは離脱します

静的ブログですとドメイン代しか必要ありませんので安いです

静的なhtml,js,cssを置くだけなので、無料で掲載できる場所はいくらでもあります

代表的なのは　GitHub Pages　他にもいっぱいあります　

Vercel、Surge、Netlify、Render

「年間１万円ほどの経費削減効果」があります

じゃあ静的ブログ はなんでみんな使ってないの？

コンパイルとかしますので、少し技術がいるからです　markdownで文章を書いて、コンパイルして gitでアップロードします

たぶんgitがＣＬＩで使える人なら覚えられると思います　一度覚えたらこっちの方が楽です

このブログのデザインでよければ、ぼくに発注してください　１万円で請け負います

何が資産か？

サーバーは借り物　いつでも移動できるよう整理しておきましょう

ブログは文章と画像データおよびブログプログラム、稼げるプログラムも資産です

またブログを書くための細かいツールやサイトもあなたにとっては資産です

自分のドメインも資産ですね　育ちますので　ドメイン育てましょう

独自ドメイン.com を育てよう!

これからは、自分の独自ドメインを強くすることを意識しましょう

これがWEBビジネスの本質です　自分を育てることと同義です　お金も労力もまずはそこに集中投資しましょう

よそのドメインを強くする行為はこれからはありえないです

例えば　無料だからといってgithub.io のサブドメインを強くするのはやめましょう

もっといえば、 youtube.com facebook.com ・・・すべてですがこれらはメディアです

バックエンドのキャッシュポイントは自分のサイトなんです

無料サーバレスで稼げばいいよ

VPSなら５００円程度 ですが、 あれは保守が大変ですので借りるのはやめましょう

サーバー管理はクラウド運営でもっしないかぎり保守が負債でしかないです

また、なにか稼げる処理ができたとして、プログラムを走らせたいだけなら、無料サーバレスで十分です

Vercel、GitHub Pages、Netlify、Render、Surge、firebase

これらの知識は相手のサービスが続くなら資産です　ぜひ見極めて習得しましょう

あとクラウドの無料枠もあります gcp aws ですが　これらはほぼ企業向けの技術です

個人単位では学習コスト以上の効果はまず期待できないと感じています

労働者として自分を企業に売るなら大いにありですが、個人なら不要かと

３０ぐらい超えてきたら　クラウドでもありですが

  • プライバシーポリシーの設置
  • カテゴリすべてに記事
  • プロフィール、プロフィール画像の設定
  • アナリティクスのトラッキングIDの取得と設置
  • Google Search Console への登録
